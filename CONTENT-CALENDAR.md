## Content Production Workflow

### Weekly Process
1. **Monday:** Outline approval
2. **Tuesday-Wednesday:** Draft creation
3. **Thursday:** Review and edit
4. **Friday:** Publish + schema markup

### Quality Checklist
- [ ] Primary keyword in H1 and first 100 words
- [ ] 2-3 internal links to existing content
- [ ] Article schema implemented
- [ ] FAQ schema if applicable
- [ ] Meta description under 160 characters
- [ ] OG image created
- [ ] CTA included (call booking or lead magnet)
